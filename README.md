# OnePlus-Nord-N10-5G-Custom-Development-Docs
Este repositorio contiene los archivos esenciales y la documentación técnica para la implementación de Kali NetHunter en el OnePlus Nord N10 5G (Codename: billie). El objetivo es proporcionar un entorno de auditoría de seguridad estable, superando las limitaciones de las particiones dinámicas y los bloqueos de solo lectura de Android 10/11.

Key Components

    Modified Boot Image: Parcheado con Magisk para acceso Root persistente y compatibilidad con el Chroot de NetHunter.

    TWRP Recovery: Versión optimizada para billie. Nota: Incluye solución para el problema de falta de respuesta táctil mediante comandos externos.

    Kali Boot Animation: Animación de inicio personalizada "Kali Dragon" para reemplazar el logo de operadora (Telcel/T-Mobile).

    Advanced Scripts: Herramientas para el montaje de particiones /system y /product en modo lectura/escritura (R/W).
